# CryptoViz
CryptoViz: crypto-currency market value visualization

## Dataset: 
crypto-market dataset from Kaggle

## Development Environment:
##### Front-end: 
HTML, CSS, Materialize

##### Interactivity:
Javascript and JQuery

##### Backend:
Flask, Python3

##### Visualization: 
Chart.js and Plotly.js

##### Data Manipulation:
pandas in Python3
